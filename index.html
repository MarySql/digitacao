Guia Amigável do Teclado

Pense nestas teclas como "botões de poder". Sozinhas, elas não fazem nada, mas quando combinadas com outras, ativam funções especiais!

Shift

Para Letras Maiúsculas: Segure Shift e pressione uma letra para torná-la maiúscula instantaneamente. Solte para voltar ao normal.

Para Símbolos Especiais: Olhe para as teclas de número (1, 2, 3...). Viu os símbolos em cima deles, como ! e @? Segure Shift para digitá-los.

Ctrl (Control)

É a tecla de "Comandos Rápidos" (atalhos). Ao segurá-la com outra letra, você dá uma ordem direta ao computador.

Combinações Famosas:
- Ctrl + C: Copia o que estiver selecionado.
- Ctrl + V: Cola o que você copiou.
- Ctrl + Z: Desfaz sua última ação. É o "botão de voltar no tempo" se cometer um erro!

Alt

A principal "magia" desta tecla no dia a dia é o atalho Alt + Tab. Segure Alt e dê um toque em Tab para ver todas as suas janelas abertas e alternar rapidamente entre elas.

AltGr

Em teclados brasileiros, algumas teclas têm 3 símbolos. A tecla AltGr serve para acessar esse terceiro símbolo, que geralmente fica no canto inferior direito da tecla (como ¹, ², ³, § ou / na tecla do Q).

Caps Lock (Fixa)

É como um interruptor para o "modo de grito". Pressione uma vez e TUDO O QUE VOCÊ DIGITAR FICARÁ EM MAIÚSCULAS. Pressione de novo para desligar. Uma luzinha no teclado geralmente acende para avisar que ela está ativa.

Tab

Extremamente útil para preencher formulários (cadastros, login, etc.). Em vez de usar o mouse para clicar no próximo campo, apenas pressione Tab para pular diretamente para ele.
